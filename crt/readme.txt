Old style text mode in browser made by Javascript and DOM manipulation!
(Rows are DIV element in main DIV container. 
In any row, SPAN elements contains omogeneous (same attributes) characters.
The render routine destroy and re-create every row, only if row changed.
Character attributes handled: background color, foreground color, blink 
state, underline state, and link.)
Free to any use, cross-platform (to test) / cross-browser old style 437 
code page monospaced bios font is provided also.
With ANSI, Keyboard and Mouse support!
